# TrueDeal Browser Extension

Check if it's actually a good deal — directly from Amazon or Myntra.

## Install (Developer Mode)

1. Open Chrome → `chrome://extensions`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the `extension/` folder

## Usage

1. Open any Amazon or Myntra product page
2. Click the TrueDeal icon in your toolbar
3. Click "Check Deal"

The extension scrapes the current page, saves it to the TrueDeal database, and shows you the deal verdict, price history stats, and whether it's worth buying now.

## Icons

Add `icon48.png` and `icon128.png` to this folder (48×48 and 128×128 px).
