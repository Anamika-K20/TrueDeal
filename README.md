# TrueDeal

A price intelligence system that tracks product prices over time, analyzes historical trends, and identifies potentially fake discounts.

## Structure

- `frontend/` — UI for browsing products and viewing price history
- `backend/scraper/` — Web scraping logic to collect price data
- `backend/api/` — REST API to serve data to the frontend
- `backend/db/` — Database models and migrations
- `backend/analysis/` — Discount detection and price trend analysis
